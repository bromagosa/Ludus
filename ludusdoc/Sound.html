id

Returns the identifier of the sound, which is the file name of the source audio file, minus the extension.

loop

Starts to play the sound in loop mode.

pause

Pauses the sound.

play

Plays the sound.

src

Returns the URL of the audio file associated with the sound.

src:

Sets the URL of the audio file associated with the sound to the one given by the argument. 

self src: 'path/to/audioFile.ogg'.

stop

Stops the sound.
