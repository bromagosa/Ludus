addFrameGroupNamed:origin:size:frameCount:

Adds a new frame group to the sprite, the name of which is given by the first argument. The second argument defines the origin point of the framegroup, the next one its size in pixels (width @ height), and the last argument defines how many frames does this frame group comprehend. 

self
    addFrameGroupNamed: 'walking'
    origin: 0 @ 100
    size: 60 @ 50
    frameCount: 12.
