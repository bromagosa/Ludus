atLastFrame

Returns true if the current frame is the last one in the group.

currentFrameNumber

Returns the position of the current frame in the group.

frameRate

Returns the rate at which the frames are cycled. The smaller the value, the faster the animation.

frameRate:

Sets the rate at which the frames are cycled to the value given by the argument. The smaller the value, the faster the animation. 

self frameRate: 2.

height

Returns the height, in pixels, of any frame in the group.

identifier

Returns the string identifier of the frame group. Usually, it will have been set by the programmer to a string with some relationship to the state the frame group represents, like 'running', 'jumping' or 'goingLeft'.

identifier:origin:frameSize:frameCount:

Sets at once the identifier, coordinates of the top left corner, size of the frames and number of frames of the framegroup.

nextFrame

Sets the next frame of the group to the current frame.

offset

Returns the horizontal coordinate of the origin of the current frame.

origin

Returns the coordinates of the left top corner of the frameGroup.

width

Returns the width, in pixels, of any frame in the group.
